================================================================================
1) SALIN / DOWNLOAD PROYEK LALU TARUH DI FOLDER TUJUAN
================================================================================
1. Download/copy seluruh folder proyek "drive" (zip atau via Git).
2. Letakkan ke: C:\capstone\drive   (boleh lokasi lain, sesuaikan perintah di bawah)

Struktur minimal yang harus ada:
  C:\capstone\drive\
    app.py                <-- FastAPI
    requirements.txt
    data\                 <-- folder data JSON
      daftar_mobil.json   <-- WAJIB
      Retail_2020.json    <-- OPSIONAL
      ...
      Wholesale_2020.json <-- OPSIONAL
      ...
    package.json          <-- Next.js di root (single-root)
    .env.local            <-- (dibuat sendiri; lihat langkah 3)
    app\page.tsx          <-- Halaman utama Next.js (App Router)
    next.config.js
    tsconfig.json
    (node_modules\ akan muncul setelah npm install)


================================================================================
2) MENJALANKAN BACKEND (FASTAPI)
================================================================================
Buka vscode nyalain terminal

# Buat & aktifkan virtualenv (pertama kali)
python -m venv .venv

run ini:
.\.venv\Scripts\Activate

# Install dependency (pertama kali / saat requirements.txt berubah)
pip install -r requirements.txt

# (Opsional) set lokasi data (kalau pakai .\data, langkah ini bisa dilewati)
# $env:DATA_DIR="C:\capstone\drive\data"

# Jalankan API (port bisa diganti jika bentrok)
uvicorn backend.app:app --reload 

Cek:
  http://127.0.0.1:8000/health  -> {"status":"ok"}
  http://127.0.0.1:8000/docs    -> Swagger


================================================================================
3) MENJALANKAN FRONTEND (NEXT.JS)
================================================================================
tambah terminal tpi jangan tutup terminal sebelomnye:

# Install dependency (pertama kali / saat package.json berubah)
npm install

# Jalankan dev server
npm run dev

Buka browser: http://localhost:3000

================================================================================ 6) GANTI DEVICE / SETUP DI MACOS (Macbook)
Buka Terminal (Command + Space, ketik "Terminal").

A. Reset / Bersihkan Environment Lama (Jika perlu):

Bash

# Hapus venv dan node_modules lama
rm -rf .venv node_modules package-lock.json
B. Setup Backend (Python):

Bash

# 1. Masuk ke folder project
cd /path/to/capstone/drive

# 2. Buat virtual environment
python3 -m venv .venv

# 3. Aktifkan virtual environment (Beda dengan Windows!)
source .venv/bin/activate

# 4. Install dependency
pip install -r requirements.txt

# 5. Jalankan Backend
uvicorn backend.app:app --reload
C. Setup Frontend (Next.js): Buka terminal baru (Command + T).

Bash

# 1. Masuk ke folder project
cd /path/to/capstone/drive

# 2. Install library
npm install

# 3. Jalankan Frontend
npm run dev
================================================================================ 7) KONFIGURASI FILE .ENV (WAJIB DIBUAT)
Anda perlu membuat 2 file konfigurasi terpisah.

A. Backend (.env) Buat file bernama .env di dalam folder root (C:\capstone\drive\.env) sejajar dengan app.py. Isinya:

Cuplikan kode

# Konfigurasi Ollama (Local)
OPENROUTER_API_KEY=ollama
OPENROUTER_MODEL=llama3.1
# Jika nanti mau pakai API berbayar, ganti key dan modelnya di sini
B. Frontend (.env.local) Buat file bernama .env.local di dalam folder root (C:\capstone\drive\.env.local). Isinya:

Cuplikan kode

# Agar Next.js tahu alamat backend FastAPI
NEXT_PUBLIC_API_BASE=http://127.0.0.1:8000
# Mode debug (opsional, set 1 untuk nyalakan DevPanel)
NEXT_PUBLIC_DEBUG=0
================================================================================ 8) PERSIAPAN OLLAMA (AI ENGINE)
